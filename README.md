# Retail Sales Analytics Project

This is a complete beginner-friendly data analytics portfolio project. It includes a generated sales dataset, repeatable analysis scripts, chart outputs, an executive summary, and an interactive dashboard.

## Project Structure

```text
Retail Sales Analytics/
  data/
    sales_data.csv
    sales_data.js
  outputs/
    monthly_revenue.svg
    profit_by_category.svg
    revenue_by_region.svg
    summary.json
  reports/
    executive_summary.md
  scripts/
    generate_data.js
    analyze.js
  index.html
  script.js
  styles.css
```

## Questions Answered

- Which regions generate the most revenue?
- Which categories create the most profit?
- How do sales change month by month?
- Which channel has the weakest margin?
- What actions should the business take next?

## How To Run

Use Node.js from a terminal inside this folder:

```powershell
node scripts/generate_data.js
node scripts/analyze.js
```

Then open `index.html` in a browser to use the dashboard.
